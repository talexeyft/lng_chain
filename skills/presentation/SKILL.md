# presentation

## Overview

Навык для создания презентаций в формате `.pptx` на основе задачи пользователя.

## Instructions

1. Уточни цель и аудиторию презентации из запроса.
2. Подготовь структуру:
   - титульный слайд,
   - 3-8 содержательных слайдов,
   - финальный слайд с выводами.
3. Для каждого слайда сформируй:
   - `title`,
   - `bullets` (3-6 коротких пунктов).
4. Сгенерируй JSON и передай его в скрипт `skills/presentation/create_presentation.py` через `execute_python`.
5. Сохрани результат в файл `.pptx` с понятным именем.
6. Верни путь к созданному файлу и краткий список слайдов.

## Notes

- Держи один слайд = одна ключевая мысль.
- Избегай длинных абзацев: только короткие тезисы.
- Если данных мало, сначала предложи черновую структуру, затем уточни детали.
