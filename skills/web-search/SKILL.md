## Overview
Этот навык предназначен для быстрого поиска информации в интернете с помощью Tavily API. Он особенно полезен, когда нужно:
- Найти актуальные данные и исследования
- Проверить информацию
- Сравнить несколько источников
- Получить ссылки на первоисточники

## Instructions
1. Используй `tavily_search` с clear and specific query
2. Обработай результаты: выдели основные точки и проверь релевантность источников
3. Верни структурированный ответ с ключевой информацией и ссылками
4. Предпочтительно используй официальные источники (академические, технические спецификации, документация компаний)

## Example Workflow

### Query: "eDRX in NB-IoT LTE-M power saving mechanisms"
1. Search for technical documentation
2. Look for 3GPP specifications, white papers
3. Compare information from multiple reliable sources
4. Summarize findings with links

## Best Practices
- Задавай конкретные и детальные поисковые запросы
- Ищи официальные документы и технические спецификации
- Проверяй дату публикации для актуальности
- Сравнивай информацию из нескольких источников
- Всегда предоставляй ссылки на источники
