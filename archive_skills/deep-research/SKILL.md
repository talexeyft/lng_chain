# deep-research

## Overview

Навык для задач, где нужен не быстрый ответ, а полноценное исследование по теме.

## Instructions

1. Сначала сформируй план исследования:
   - ключевые вопросы,
   - критерии качества источников,
   - ожидаемый формат результата.
2. Выполни серию поисков через `tavily_search_results_json`:
   - обзорные запросы,
   - уточняющие запросы по каждому подвопросу,
   - проверка альтернативных точек зрения.
3. Открой важные страницы через `fetch_url` и извлеки факты.
4. Проверяй достоверность:
   - сравни минимум 2 независимых источника для ключевых тезисов,
   - отмечай уровень уверенности.
5. Сформируй результат в структуре:
   - Executive summary,
   - Findings (по разделам),
   - Risks / ограничения данных,
   - Sources (список URL).

## Output template

- Executive summary: 5-10 предложений.
- Findings: разделы по подвопросам с фактами и контекстом.
- Risks: что не удалось подтвердить и почему.
- Sources: нумерованный список ссылок.
