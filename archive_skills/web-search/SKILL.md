# web-search

## Overview

Навык для быстрого поиска свежей информации в интернете и подготовки ответа с источниками.

## Instructions

1. Сформулируй 2-4 точных поисковых запроса на основе задачи пользователя.
2. Для каждого запроса используй `tavily_search_results_json`.
3. Выбери 3-6 наиболее релевантных источников, исключая дубликаты.
4. При необходимости прочитай страницу глубже через `fetch_url`.
5. Верни:
   - краткий итог,
   - ключевые факты,
   - список источников (название + URL).

## Quality bar

- Приоритет у свежих и авторитетных источников.
- Если данные противоречат друг другу, явно укажи расхождения.
- Не выдавай непроверенные предположения как факт.
